# Simple Interest Calculator

## Project Description
This is a simple web application that calculates Simple Interest based on the given Principal Amount, Rate of Interest, and Time.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Features
- User can enter Principal amount
- User can enter Rate of Interest
- User can enter Time period
- Calculates Simple Interest instantly
- Displays the result on screen

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Enter the Principal amount
2. Enter the Rate of Interest
3. Enter the Time in years
4. Click the Calculate button
5. The Simple Interest will be displayed

## Example
Principal = 1000  
Rate = 5%  
Time = 2 years  

Simple Interest = 100
